A bash script to interact with the forwardemail.net API.

```
Usage: feapi <list|create>

 Commands
    list-aliases <domain>                               -List all aliases for domain
    view-alias <email alias>                            -Retrive specific domain alias
    create-alias <email alias> <destination-emails>     -Creates an alias with comma separated destination emails
    delete-alias <email alias>                          -Deletes an alias
    tests                                               -List all test codes
    debug                                               -Debug mode
```
